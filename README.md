# Boost.Asio Port Scanner

A simple and efficient TCP port scanner implemented in **C++** using the **Boost.Asio** networking library.  
This project demonstrates asynchronous networking, socket programming, and basic concurrency concepts.

---

## Features

- Asynchronous TCP connect scanning
- Fast and lightweight implementation
- Configurable target host and port range
- Uses Boost.Asio for non-blocking I/O
- Clean and readable C++ codebase

---

## Technologies Used

- C++ (C++17 or later)
- Boost.Asio
- CMake (for building)

---

## Requirements

- C++ compiler with C++17 support (GCC, Clang, MSVC)
- Boost library (version 1.70 or higher)
- CMake 3.10+
